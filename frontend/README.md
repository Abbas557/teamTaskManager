# Team Task Manager Frontend

React frontend for the Team Task Manager app.

## Setup

```bash
npm install
npm run dev
```

The app runs at:

```txt
http://localhost:5173
```

The default API base URL is:

```txt
http://localhost:3000/api
```

Override it with:

```env
VITE_API_BASE_URL=http://localhost:3000/api
```

## Demo Login

After the backend seed runs:

```txt
admin@example.com / Admin1234
member@example.com / Member1234
```
